Setup guide

To enable central quote logging:
1. Create a Google Sheet at sheets.google.com
2. Go to Extensions → Apps Script
3. Paste the webhook code (see SETUP_INSTRUCTIONS.md)
4. Deploy as Web App → copy URL
5. Paste URL above and tap Save

Quotes will then appear in the sheet accessible to your Sales Manager.
